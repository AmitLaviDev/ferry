Verify this is the case for AWS provider 6.x — if not, remove assume_role and let the bootstrap run with ambient credentials.)

Best approach for provider block:
```hcl

  region = var.region

  default_tags {

      ManagedBy = "terraform"
      Project   = "ferry"
    }
  }
}
```
