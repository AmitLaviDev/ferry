<task type="checkpoint:human-action" gate="blocking">
  <name>Task 1: Register GitHub App and populate secrets</name>
  <action>
Follow the setup runbook at `docs/setup-runbook.md` Steps 1-4. Exact instructions below for reference.

**Step 1 -- Generate a webhook secret:**
```bash
openssl rand -hex 20
```
Save this value.

**Step 2 -- Register the GitHub App** at https://github.com/settings/apps/new :
- **Name:** Choose a name (e.g., "Ferry" or "Ferry-Staging")
- **Homepage URL:** Your ferry repo URL
- **Webhook URL:** The Lambda Function URL from Plan 01
- **Webhook secret:** The hex string from Step 1
- **Repository permissions:** Contents=Read, Pull requests=Read&Write, Checks=Read&Write, Actions=Write
- **Events:** Push (checked)
- **Installation scope:** Only on this account
- Click **Create GitHub App**

**Step 3 -- Capture App ID and generate private key:**
- Note the **App ID** from the app settings page
- Under **Private keys**, click **Generate a private key** and save the `.pem` file

**Step 4 -- Install the App on the ferry repo:**
- Go to the app settings > **Install App** (left sidebar)
- Select the account > **Only select repositories** > pick the `ferry` repo
- Click **Install**
- Note the **Installation ID** from the URL: `https://github.com/settings/installations/{INSTALLATION_ID}`

**Step 5 -- Populate Secrets Manager:**
```bash
aws secretsmanager put-secret-value \
  --secret-id ferry/github-app/app-id \
  --secret-string "<APP_ID>"

aws secretsmanager put-secret-value \
  --secret-id ferry/github-app/private-key \
  --secret-string "$(cat /path/to/your.private-key.pem)"

aws secretsmanager put-secret-value \
  --secret-id ferry/github-app/webhook-secret \
  --secret-string "<WEBHOOK_SECRET>"
```

**Step 6 -- Verify secrets are populated:**
```bash
aws secretsmanager get-secret-value --secret-id ferry/github-app/app-id --query SecretString --output text
aws secretsmanager get-secret-value --secret-id ferry/github-app/private-key --query 'SecretString' --output text | head -c 50
aws secretsmanager get-secret-value --secret-id ferry/github-app/webhook-secret --query SecretString --output text
```

**Step 7 -- Set Installation ID via Terraform (Option B -- permanent):**

Create or update `iac/aws/staging/us-east-1/ferry_backend/terraform.tfvars`:
```hcl
installation_id = "<INSTALLATION_ID>"
```

Then apply:
```bash
terraform -chdir=iac/aws/staging/us-east-1/ferry_backend apply -input=false
```

**Step 8 -- Set GHA repo secret:**
- Go to ferry repo on GitHub > **Settings** > **Secrets and variables** > **Actions**
- New repository secret: **Name** = `AWS_DEPLOY_ROLE_ARN`, **Value** = the self-deploy role ARN from Plan 01
  </action>

</task>
