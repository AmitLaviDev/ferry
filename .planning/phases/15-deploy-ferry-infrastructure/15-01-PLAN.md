<task type="checkpoint:human-action" gate="blocking">
  <name>Task 1: Run bootstrap script and apply remaining Terraform projects</name>
  <action>
Execute the full Terraform apply chain in dependency order. The bootstrap script handles the first two projects; the remaining three are applied manually.

**Step 1 -- Run bootstrap script (S3 state + ECR + placeholder image):**
```bash
./scripts/bootstrap.sh
```
This is idempotent -- safe to re-run. It creates the S3 state bucket, ECR repo, and pushes the placeholder image.

**Step 2 -- Apply OIDC provider (global):**
```bash
terraform -chdir=iac/global/cloud/aws/oidc init
terraform -chdir=iac/global/cloud/aws/oidc plan -input=false
# Review plan output -- should create 1 resource (aws_iam_openid_connect_provider)
terraform -chdir=iac/global/cloud/aws/oidc apply -input=false
```

**Step 3 -- Apply shared IAM + Secrets (staging):**
```bash
terraform -chdir=iac/aws/staging/shared init
terraform -chdir=iac/aws/staging/shared plan -input=false
# Review plan output -- should create IAM roles, policies, and Secrets Manager containers
terraform -chdir=iac/aws/staging/shared apply -input=false
```

**Step 4 -- Apply backend (Lambda + DDB + Function URL):**
```bash
terraform -chdir=iac/aws/staging/us-east-1/ferry_backend init
terraform -chdir=iac/aws/staging/us-east-1/ferry_backend plan -input=false
# Review plan output -- should create Lambda, Function URL, DynamoDB table, CloudWatch log group
terraform -chdir=iac/aws/staging/us-east-1/ferry_backend apply -input=false
```

**Step 5 -- Save the Function URL and self-deploy role ARN for later steps:**
```bash
terraform -chdir=iac/aws/staging/us-east-1/ferry_backend output -raw lambda_function_url
terraform -chdir=iac/aws/staging/shared output -raw gha_self_deploy_role_arn
```
Save both values -- they are needed in Plan 02.

**IMPORTANT:** If any `terraform plan` shows unexpected changes (drift, resource recreation), STOP and investigate before applying. Do not blindly apply.
  </action>
  <resume-signal>Confirm all five terraform projects applied successfully. Paste the Function URL and self-deploy role ARN.</resume-signal>
</task>
