  <action>
    Create the shared data contract that decouples Ferry App and Ferry Action development.

    **constants.py:**
    - `SCHEMA_VERSION = 1` — payload schema version
    - `ResourceType` enum (StrEnum): `LAMBDA = "lambda"`, `STEP_FUNCTION = "step_function"`, `API_GATEWAY = "api_gateway"`
    - `RESOURCE_TYPE_WORKFLOW_MAP` dict mapping ResourceType to workflow dispatch type names (e.g., `"lambdas"`, `"step_functions"`, `"api_gateways"`)

    **errors.py:**
    - `FerryError(Exception)` — base error
    - `WebhookValidationError(FerryError)` — signature validation failures
    - `DuplicateDeliveryError(FerryError)` — dedup detected duplicate (not really an error, used for flow control logging)
    - `GitHubAuthError(FerryError)` — JWT/token exchange failures
    - `ConfigError(FerryError)` — ferry.yaml parsing failures (for Phase 2)

    **models/dispatch.py** (per user's locked decision — discriminated union, minimal payload):
    - `LambdaResource(BaseModel)`: resource_type: Literal["lambda"], name: str, source: str, ecr: str
    - `StepFunctionResource(BaseModel)`: resource_type: Literal["step_function"], name: str, source: str
    - `ApiGatewayResource(BaseModel)`: resource_type: Literal["api_gateway"], name: str, source: str
    - `Resource` = Annotated Union with Field(discriminator="resource_type")
    - `DispatchPayload(BaseModel)`: v: int = 1 (schema version), resource_type: str, resources: list[Resource], trigger_sha: str, deployment_tag: str, pr_number: str = "" (empty if not PR)
    - Add `model_config = ConfigDict(frozen=True)` to make payloads immutable

    **models/webhook.py:**
    - `WebhookHeaders(BaseModel)`: event_type: str, delivery_id: str, signature: str — extracted from raw headers
    - `PushEvent(BaseModel)`: Minimal fields needed for Phase 1 processing — ref: str, before: str, after: str, repository: Repository, pusher: Pusher
    - `Repository(BaseModel)`: full_name: str, default_branch: str
    - `Pusher(BaseModel)`: name: str

    **models/__init__.py:** Re-export all models for convenient imports.
    **utils/__init__.py:** Re-export key types: DispatchPayload, Resource, LambdaResource, StepFunctionResource, ApiGatewayResource, PushEvent, WebhookHeaders, ResourceType, SCHEMA_VERSION.

    **tests/test_utils/test_dispatch_models.py:**
    - Test DispatchPayload with LambdaResource validates correctly
    - Test discriminated union rejects unknown resource_type
    - Test DispatchPayload with mixed resource types (should fail — resources in one payload should be same type per per-type dispatch model)
    - Test schema version defaults to 1
    - Test pr_number defaults to empty string
    - Test frozen model rejects mutation

    **tests/test_utils/test_webhook_models.py:**
    - Test PushEvent parses a real GitHub push event payload structure
    - Test WebhookHeaders extraction from raw header dict
    - Test Repository and Pusher sub-models
  </action>
