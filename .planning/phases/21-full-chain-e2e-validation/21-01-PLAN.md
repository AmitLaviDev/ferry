  <action>
Modify the Lambda handler in the test repo to echo back the input payload, push it, and verify Ferry deploys it.

**Step A: Modify Lambda handler**
```bash
cd /path/to/ferry-test-app
```
Edit `services/hello-world/main.py` (or `lambdas/hello-world/main.py` — use whichever path exists) to echo input:
```python

    return {
        "source": event.get("source", "unknown"),
        "message": "hello from ferry",
    }
```

**Step B: Push the change**


**Step C: Verify Ferry processes the webhook** (within 30s)


**Step D: Verify GHA workflow completes** (wait up to 5 min)
```bash
gh run list -R AmitLaviDev/ferry-test-app --limit 3
# Get the run ID for the lambdas workflow
gh run view -R AmitLaviDev/ferry-test-app <RUN_ID>
```

**Step E: Direct invoke to verify echo**
```bash

  --cli-binary-format raw-in-base64-out \
```

Expected response: `{"source": "ferry-e2e-test", "message": "hello from ferry"}`
